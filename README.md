# aegiscrypto

> Post-quantum cryptographic operations, certificate validation, and secure hashing at scale

## Overview

aegiscrypto is a production-grade security system designed for protecting critical infrastructure and citizen data. It combines advanced cryptographic techniques with intelligent monitoring to ensure comprehensive security coverage.

## Features

- Post-quantum key generation — CRYSTALS-Kyber and Dilithium key pairs
- Digital signatures — Quantum-resistant signing and verification
- Key encapsulation — Secure shared secret establishment
- Certificate management — Full lifecycle with chain validation
- High-performance hashing — SHA-256/384/512, BLAKE2b with batch processing
